# qRobotRemote
Qt Quick application for controlling a remote robot via Bluetooth via on screen joystick.

## Protocol

The communication over RFCOMM with a paired device:

> [Direction] + [Power]

## Direction values

 * **U** Up
 * **D** Down
 * **L** Left
 * **R** Right
 * **S** Stop

## Power

Range [1-100]

0 to stop
